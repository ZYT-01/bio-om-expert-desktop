## 一、前置依赖安装

无论 macOS 还是 Windows，应用运行需要以下三个基础环境：

### 1.1 安装 Node.js（必须）

| 平台 | 下载地址 |
|------|----------|
| macOS | https://nodejs.org/ → 下载 macOS 安装包 (.pkg) |
| Windows | https://nodejs.org/ → 下载 Windows 安装包 (.msi) |

选择 **LTS（长期支持版）**，一路默认安装即可。

验证安装：
```bash
node --version    # 应显示 v18.x 或更高
npm --version     # 应显示 9.x 或更高
```

### 1.2 安装 Claude Code CLI（必须）

打开终端（Windows 用 PowerShell 或 CMD），执行：
```bash
npm install -g @anthropic-ai/claude-code
```

验证安装：
```bash
claude --version   # 应显示版本号
```

### 1.3 配置 Anthropic API Key（必须）

```bash
# macOS / Linux — 写入 ~/.zshrc 或 ~/.bashrc
echo 'export ANTHROPIC_API_KEY="sk-你的密钥"' >> ~/.zshrc
source ~/.zshrc

# Windows PowerShell
[Environment]::SetEnvironmentVariable("ANTHROPIC_API_KEY", "sk-你的密钥", "User")
```

### 1.4 安装 Python3 + python-docx（必须）

**macOS:**
```bash
# macOS 通常自带 Python3，如果没有：
brew install python3

pip3 install python-docx
```

**Windows:**
1. 从 https://python.org/ 下载安装
2. 安装时勾选 "Add Python to PATH"
3. 打开 CMD 执行：
```cmd
pip install python-docx
```

## 二、一键环境配置

前置依赖装好后，运行 setup 脚本自动安装 Skills 和 Python 依赖：

**macOS:**
```bash
cd release/
bash setup.sh
```

**Windows:**
```cmd
cd release\
setup.bat
```

脚本会自动完成：
- 检测 Node.js / Claude CLI / Python3 是否就绪
- 安装 `python-docx`
- 将 5 个 Skills 复制到 `~/.claude/skills/`

---

## 三、安装应用

### macOS

1. 双击 `Bio-OM Expert_1.0.0_aarch64.dmg`
2. 将 `Bio-OM Expert.app` 拖入 `Applications` 文件夹
3. 首次打开时：右键点击 → **打开**（跳过 Gatekeeper 未签名提示）
4. 应用窗口出现，状态栏显示 "✓ 就绪"

### Windows

1. 双击 `Bio-OM Expert_1.0.0_x64.msi`
2. 按向导完成安装
3. 从开始菜单或桌面快捷方式启动
4. 如弹出 SmartScreen 警告，点击 "更多信息" → "仍要运行"

## 四、验证安装

启动应用后，在输入框中输入：
```
SOD抗氧化机制科普
```

点击 **▶ 运行**，应该看到：
1. 进度条从"分析需求..."开始推进
2. 日志流实时显示内容生成过程
3. 完成后右侧出现文件预览面板
4. 输入框下方出现 "📂 全部文件" 和 "📄 Word 文档" 按钮

## 五、常见问题

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 点运行没反应 | Claude CLI 未安装或未配置 API Key | 终端执行 `claude --version` 确认 |
| "Claude CLI 启动失败" | 环境变量未生效 | 重启终端或重新登录 |
| 没有生成 Word 文档 | python-docx 未安装 | `pip3 install python-docx` |
| macOS "无法验证开发者" | 未签名 | 右键点击 → 打开 |
| Windows SmartScreen 拦截 | 未签名 | 更多信息 → 仍要运行 |
| 进度条卡住不动 | 网络问题或 API 限流 | 等待或重试 |

## 六、更新方式

新版本发布后：
1. 从 GitHub Releases 下载最新 `.dmg` / `.msi`
2. 覆盖安装（无需重新配置环境）
3. Skills 如有更新，重新运行 `setup.sh` / `setup.bat`

## 七、技术支持

- **项目地址**: bio-om-expert
- **Skills 文档**: `~/.claude/skills/` 各目录下的 SKILL.md
