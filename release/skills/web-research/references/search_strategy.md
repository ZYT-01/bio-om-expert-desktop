# 搜索策略与技巧

> 用途: web-research 技能 · 参考文档
> 适用场景: 生物科技行业全网信息搜索

## 一、搜索原则

### 1.1 覆盖度优先
- 每个主题至少覆盖 3 类搜索源（学术/行业/社交）
- 中英文关键词各生成至少 5 个变体
- deep 模式下需追溯引用链（引文正向/反向检索）

### 1.2 时效性分层

| 主题类型 | 时效范围 | 说明 |
|----------|----------|------|
| 技术进展 | 近 2 年 | 快速迭代领域 |
| 政策法规 | 近 1 年 | 保持最新合规 |
| 市场分析 | 近 3 年 | 趋势需时间验证 |
| 基础研究 | 近 5 年 | 经典文献仍可参考 |

### 1.3 可信度优先
- 优先收录: 同行评议期刊、官方机构发布、上市公司披露
- 有条件收录: 行业媒体、分析师报告（需标注立场）
- 谨慎收录: 社交媒体、匿名来源（仅作趋势参考）

## 二、关键词构造策略

### 2.1 基础公式
```
{核心技术} + {应用场景} + {时间限定/地域限定}
```
示例: `AAV + 基因治疗 + 商业化 + 2024|2025`

### 2.2 中英文对照搜索
- 中文搜索在中国市场/政策类信息有优势
- 英文搜索在学术/全球市场类信息有优势
- 重要主题需双语交叉验证

### 2.3 同义扩展
| 概念 | 中文变体 | 英文变体 |
|------|----------|----------|
| 基因治疗 | 基因疗法/基因药物/基因替代治疗 | gene therapy/gene medicine/gene replacement |
| 商业化 | 上市/获批/市场准入/定价 | commercialization/approval/market access/pricing |

## 三、搜索源优先级

| 优先级 | 源类型 | 代表 | 适用场景 |
|--------|--------|------|----------|
| P0 | 监管机构 | NMPA CDE, FDA, EMA | 审批进展、政策法规 |
| P0 | 学术期刊 | Nature Biotech, Cell, Science | 技术突破、临床数据 |
| P1 | 行业数据 | 医药魔方、药智网 | 市场数据、竞争格局 |
| P1 | 预印本 | BioRxiv, MedRxiv | 前沿研究 |
| P2 | 行业媒体 | 生物谷、医麦客 | 行业动态、企业动向 |
| P2 | 公司官方 | 官网、投资者关系 | 管线进展、财务数据 |
| P3 | 社交媒体 | 微信公众号、知乎 | 行业讨论、专家观点 |

## 四、去重与过滤

### 4.1 去重规则
1. URL 去重（精确匹配）
2. 标题相似度 > 0.85 → 保留发布时间最早的来源
3. 正文相似度 > 0.80 → 合并为一条，标注多来源

### 4.2 过滤规则
- 过滤: 纯广告/推广文、无实质内容的聚合页
- 降权: 发布时间超过时效范围的内容
- 标记: 潜在利益冲突来源（如企业自宣稿）

## 五、质量检查清单

- [ ] 是否覆盖了中英文双语来源？
- [ ] 是否包含了至少一个一手数据来源？
- [ ] 是否有时间标注（发布时间/数据时间）？
- [ ] 是否标注了来源可信度？
- [ ] 是否标记了信息矛盾之处？
- [ ] 是否过滤了明显的商业推广内容？
