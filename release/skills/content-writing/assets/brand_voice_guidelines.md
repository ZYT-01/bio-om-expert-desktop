# 品牌语调指南

> 用途: content-writing 技能 · 品牌内容创作参考
> 适用场景: 撰写推文和视频脚本时保持品牌语调一致

## 一、品牌人格

| 维度 | 描述 |
|------|------|
| **角色** | 值得信赖的行业专家，而非推销员 |
| **语气** | 专业但不冰冷，深度但不晦涩 |
| **关系** | 与读者是同行对话，不是师生授课 |

## 二、语言风格

### 2.1 推荐表达

| 避免 | 推荐 |
|------|------|
| "我们的产品非常好" | "数据显示，该方案可提升收率30%" |
| "行业领先" | "已有12家CDMO企业采用该方案" |
| "独一无二" | "目前全球仅3家企业具备该工艺能力" |
| "效果显著" | "对比测试中，空壳率从15%降至3%以下" |

### 2.2 禁用词汇

- 绝对化用语：最佳、唯一、第一、100%、零风险
- 过度承诺：保证、必定、绝对安全
- 贬低竞品：明确攻击具体竞品名称

### 2.3 推荐用词

- 专业克制：数据显示、研究表明、实际案例表明
- 客观中立：在特定条件下、基于当前数据、有待进一步验证
- B端价值导向：降低成本、提升效率、降低风险、合规达标

## 三、目标受众语气映射

| 受众 | 语气调整 | 关键词偏好 |
|------|----------|-----------|
| 研发/QC人员 | 技术深度优先，用数据说话 | 工艺参数、质量标准、检测方法 |
| 采购/供应链 | 突出成本和效率，简洁直接 | 成本优势、供应稳定性、交付周期 |
| 企业高管 | 战略价值优先，关注趋势和风险 | 行业趋势、竞争壁垒、长期价值 |
| 投资者 | 市场潜力+技术壁垒，逻辑严密 | 市场规模、技术优势、增长预期 |

## 四、情感曲线

技术推文的情感节奏应是：

```
好奇（标题） → 共鸣（痛点） → 惊讶（数据） → 理解（原理） → 信服（案例） → 行动（CTA）
```

避免：
- 过度煽情（不适合B端场景）
- 平铺直叙（缺乏节奏感）
- 虎头蛇尾（结尾无力）

## 五、CTA（行动号召）规范

| 场景 | CTA示例 |
|------|---------|
| 技术推文 | "如需完整技术白皮书，欢迎留言" |
| 案例分享 | "想了解您的应用场景？联系我们做方案评估" |
| 行业趋势 | "扫码获取完整行业报告" |
