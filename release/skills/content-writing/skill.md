# Skill: content-writing（文案撰写与视频脚本制作）

## 用途
基于确认的选题和第一阶段生成的研究报告，撰写公众号推文（含标题、摘要、正文、图文互动设计）并同步改编为视频脚本（含分镜描述、AI提示词、口播稿及视频互动设计）。

## 触发条件
- 用户输入：`/content-writing --report "report.md" --topic "选题名称"`
- 示例：`/content-writing --report "./output/reports/20260626_AAV空壳率_report.md" --topic "AAV载体空壳率成因与QC解决方案"`

## 输入字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| report_content | string | 是 | 研究报告内容（路径或直接文本） |
| topic | string | 是 | 选题名称（对应报告中的主题簇） |
| target_audience | string | 否 | 目标受众（默认：生物医药企业研发/QC人员） |
| word_count | int | 否 | 推文目标字数（默认：1000-1200字） |
| video_duration | int | 否 | 视频目标时长（默认：180秒） |
| image_style | string | 否 | 配图风格偏好（默认：科学可视化/专业蓝紫调） |
| platform | enum | 否 | 视频平台类型：`short`（抖音/快手，0-30秒）或 `long`（视频号/小红书，1-3分钟），默认 `long` |

## 执行步骤

### 步骤1：报告研读与核心提炼
1. 读取研究报告全文，定位与 `topic` 相关的章节
2. 提炼以下关键要素：
   - 核心问题（客户痛点）
   - 技术原理/解决方案
   - 应用场景/案例数据
   - 价值结论

### 步骤2：推文大纲构建
调用 `scripts/outline_builder.py` 辅助构建大纲，采用 **"问题引入→技术解析→应用场景→价值总结"** 四段结构；大纲中标记配图节点位置。

### 步骤3：推文正文撰写
1. 按大纲结构逐段撰写，**在需要配图的句子后添加 `[配图建议]` 标记**
2. 调用 `scripts/word_counter.py` 实时校验字数
3. **写作金规则**：用客户语言/每段≤300字/加粗关键词/字数800-1500字

---

### 步骤4：内文配图规划与提示词生成

#### 4.1 配图节点识别
基于已完成正文，调用 `scripts/image_suggestion_generator.py` 扫描并识别需要配图的节点：

| 配图类型 | 识别特征 | 示例场景 |
|----------|----------|----------|
| **原理示意图** | 描述机制、结构、工作原理的段落 | 口溶膜吸收机制、24h闭环运作 |
| **流程步骤图** | 包含顺序词或步骤编号 | 三步优化流程、产品使用步骤 |
| **数据对比图** | 包含百分比、对比、升降表述 | 市场数据对比、效果数据 |
| **概念模型图** | 包含"由...组成""包括...部分" | 产品成分体系、四重复方 |
| **场景应用图** | 描述使用场景或操作画面 | 睡前使用场景、实验室操作 |

#### 4.2 配图需求清单生成
为每个识别到的配图节点生成结构化需求记录。

#### 4.3 画面描述撰写
为每个配图需求撰写详细画面描述（主体/构图/色彩/标注/风格）。

#### 4.4 AI绘图提示词生成
基于画面描述，按公式 `[主体] + [环境] + [风格] + [画幅] + [分辨率]` 生成英文AI绘图提示词。

#### 4.5 配图与正文关联
将生成的配图建议以 `[配图建议：ID-XXX - 说明]` 的形式插入正文对应位置。

---

### 步骤5：图文互动设计脚本生成（基于正文）

基于已完成的推文正文，为图文平台（公众号/朋友圈/小红书）生成四种互动行为的触发点设计。

**设计原则**：
- 用户不会无缘无故互动，每一次点赞/收藏/转发/评论背后都有明确的"心理收益"。
- 运营设计的本质，就是在内容之外为这四种行为分别设计"触发点"。

**生成方法**：
| 互动类型 | 心理收益 | 设计触发点方法 | 示例（生物技术B端） |
|----------|----------|---------------|---------------------|
| **点赞** | 认同感、情绪共鸣 | 在文末或文中埋入一个"认同钩子"，如"如果你也遇到过这个问题，点个赞让我知道"。 | "如果你也曾在QC报告中为空壳率头疼过，请点个赞" |
| **收藏** | 知识留存、未来使用 | 提供清单、模板、步骤总结，明确告知"值得收藏" | "建议收藏：AAV空壳率优化三步法完整版" |
| **转发** | 社交货币、利他性 | 创造"值得分享给同事/客户"的价值，如行业趋势洞察、案例解析 | "转发给正在做AAV工艺的同事，帮他少走弯路" |
| **评论** | 参与感、被重视 | 提出开放性问题、征集经验或观点，降低评论门槛 | "你在纯化中遇到的最大难题是什么？评论区聊聊" |

**输出格式**：调用 `scripts/interaction_designer.py` 生成结构化互动设计脚本，明确每个互动类型的触发位置、触发文案和心理收益说明。

---

### 步骤6：标题生成
1. 调用 `scripts/headline_generator.py` 生成候选标题
2. 筛选并优化为3-5个备选，覆盖数字型/问题型/对比型/价值型/悬念型

### 步骤7：摘要撰写
- 100字以内，包含：目标受众+核心价值+阅读收益

### 步骤8：视频脚本改编
1. 从推文核心知识点压缩为1-5分钟视频
2. 四段式结构：开场痛点(15-20%)/技术原理(35-40%)/应用案例(30-35%)/价值总结(10-15%)
3. 调用 `scripts/script_timeline_allocator.py` 计算字数与时长对应关系
4. 时长分配根据 `platform` 参数调整（short: 0-30秒，long: 1-3分钟）

### 步骤9：视频分镜与提示词撰写
1. 为每个镜头撰写画面描述（中文）+ AI提示词（英文，含画幅和风格参数）
2. 调用 `scripts/prompt_enhancer.py` 优化提示词质量

---

### 步骤10：中文口播稿生成（基于视频脚本）

在完成视频脚本和分镜后，基于脚本中的旁白/配音文案，生成一份适合真人配音的中文口播稿。

**口播稿生成规则**：

1. **字数与时长的计算**：
   - 核心公式：`总字数 ÷ 语速 = 总时长(分钟)`
   - 标准语速：**180-200字/分钟**（适用于1-3分钟的"认知占领"型内容）
   - 快语速：200-250字/分钟（适用于0-30秒的"本能反应"型短内容，如抖音快手）

2. **留白与停顿**：
   - 最终成片时长应 **略大于** 纯读稿时间（预留10%-15%的余量用于呼吸、停顿、重音）。
   - 口播稿中在关键句前后添加 `（停顿）` 或 `（重音）` 标记，帮助后期配音把握节奏。

3. **口语化改造**：
   - 将书面语转为口语：多使用短句、口语助词（"其实"、"然后呢"、"你知道吗"）。
   - 避免长定语和复杂从句，用逗号或句号切分为短句。

4. **节奏控制**：
   - 核心卖点、金句、结论处放慢语速、加重语气（标注 `[慢速+重音]`）。
   - 过渡或铺垫部分可稍快（标注 `[快速]`）。

5. **平台差异化**：
   - 若 `platform = "short"`（0-30秒）：口播稿字数控制在80-100字，语速200-250字/分钟，内容高度凝练，直击痛点。
   - 若 `platform = "long"`（1-3分钟）：口播稿字数控制在180-600字，语速180-200字/分钟，内容完整有层次。

**输出**：调用 `scripts/voiceover_generator.py` 生成一份完整的中文口播稿文本，包含节奏标记和时长估算。

---

### 步骤11：视频互动设计脚本生成（基于视频脚本和口播稿）

为视频平台（抖音/快手/视频号/小红书视频）设计互动触发点，策略与图文略有差异，更强调"在视频中嵌入互动引导"。

**设计方法**：
| 互动类型 | 触发时机 | 示例话术（融入口播稿或画面） |
|----------|----------|------------------------------|
| **点赞** | 结尾或共鸣点之后 | "觉得有用的话，点个赞支持下吧" |
| **收藏** | 结尾（沉淀价值） | "收藏起来，做实验时直接对照" |
| **转发** | 中间或结尾（利他性） | "分享给团队里负责纯化的同事，一起优化" |
| **评论** | 开头或中间（引发讨论） | "你在AAV纯化中遇到过什么坑？评论区告诉大家" |

**输出格式**：调用 `scripts/interaction_designer.py`（视频模式）为每个互动类型生成触发点描述，包括：触发位置（视频时间点）、引导话术、呈现方式（口播/字幕/贴纸）。

## 输出字段
| 字段 | 类型 | 说明 |
|------|------|------|
| topic_id | string | 选题ID |
| article_draft | object | 推文草稿（含标题列表、摘要、正文、大纲、配图建议） |
| article_interaction | object | 图文互动设计脚本（点赞/收藏/转发/评论触发点） |
| video_script | object | 视频脚本（含时长、分镜列表、完整旁白） |
| video_voiceover | string | 中文口播稿（含节奏标记、时长估算） |
| video_interaction | object | 视频互动设计脚本（点赞/收藏/转发/评论触发点） |
| article_path | string | 推文草稿文件路径 |
| script_path | string | 视频脚本文件路径 |
| suggestions_path | string | 配图建议清单文件路径 |
