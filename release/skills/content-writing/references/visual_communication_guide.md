# 科学可视化视觉传达指南

> 用途: content-writing 技能 · 参考文档
> 适用场景: 为生物科技B端内容选择配图风格和生成AI提示词时参考

## 一、生物科技视觉传达原则

### 1.1 准确性 > 美观性
- 科学内容的首要价值是准确性
- 美观服务于理解，而非装饰
- 避免为"好看"而牺牲科学严谨性

### 1.2 "解释即设计"
- 每一张配图必须回答一个具体的问题
- 如果读者需要3秒以上才理解配图的核心信息，说明设计失败
- 好的科学配图让复杂概念"一目了然"

### 1.3 一致性
- 同一推文内所有配图风格统一（色彩、字体、光照）
- 同一品牌所有推文配图风格一致
- 建立品牌视觉识别度

## 二、五类配图的视觉规范

### 2.1 原理示意图

**目标**：解释"为什么"和"如何"

**视觉规范**：
- 剖面/切面视角展现内部结构
- 使用对比色区分"正常"与"异常"状态
- 关键部位用高亮标注
- 保持3D感但避免过度复杂

**常见错误**：
- ❌ 把所有细节塞进一张图
- ❌ 标注线交叉混乱
- ✅ 一层信息一张图，复杂原理分多张展示

### 2.2 数据对比图

**目标**：用数字证明"好坏"

**视觉规范**：
- 柱状图比饼图更准确（人类不擅长比较角度）
- Before/After 左右对比最直观
- 关键数据点用品牌强调色
- Y轴从零开始（避免误导性视觉差）

**常见错误**：
- ❌ Y轴截断（制造虚假的"巨大差异"）
- ❌ 不必要的3D效果（扭曲数据感知）
- ✅ 简洁平面图表，关键数据大号标注

### 2.3 流程步骤图

**目标**：展示"先做什么，后做什么"

**视觉规范**：
- 从左到右或从上到下（符合阅读习惯）
- 步骤数 3-7 最佳（超过7步需分组）
- 步骤间用箭头连接
- 每个步骤含图标+标题+一行说明

**常见错误**：
- ❌ 步骤数量过多（超过10步）
- ❌ 箭头方向混乱
- ✅ 简洁步骤卡片，视觉节奏均匀

### 2.4 概念模型图

**目标**：展示"由什么组成，如何组织"

**视觉规范**：
- 等距/分层视角展现层级关系
- 各组件标签清晰，大小反映重要性
- 用颜色编码区分不同模块
- 类似学术期刊配图的专业感

**常见错误**：
- ❌ 所有组件一样大（无层级感）
- ❌ 标签太多，互相遮挡
- ✅ 顶层概念大而简，底层细节小而精

### 2.5 场景应用图

**目标**：展示"实际怎么用"

**视觉规范**：
- 真实现代实验室场景（非影视剧夸张风格）
- 人物操作自然，非摆拍感
- 设备/环境细节真实可信
- 柔和自然光线，不过度滤镜

**常见错误**：
- ❌ 穿白大褂但背景像厨房（不真实）
- ❌ 冰冷无情感（B端不等于没有温度）
- ✅ 真实的科研工作场景，专业而亲切

## 三、AI提示词最佳实践

### 3.1 提示词结构公式

```
[主体描述] + [环境/背景] + [光照] + [风格关键词] + [画幅] + [分辨率] + [排除项]

示例：
Cross-section of AAV viral capsid showing genome packaging,
dark blue gradient background, soft rim lighting,
scientific 3D illustration style, clean laboratory aesthetic,
16:9, 4K,
--no text, watermark, low quality, cartoon
```

### 3.2 负面提示词标准模板

```
--no text, watermark, logo, low quality, blurry, oversaturated,
cartoon, sketch, hand-drawn, messy, cluttered, unrealistic
```

### 3.3 风格关键词参考

**推荐使用**：
- scientific illustration
- photorealistic 3D render
- clean laboratory aesthetic
- professional biotechnology visualization
- clinical photography style

**避免使用**：
- anime / cartoon / sketch
- fantasy / surreal
- overly dramatic lighting
- Instagram filter effect

## 四、色彩约定

| 用途 | 色值 | 场景 |
|------|------|------|
| **品牌主色** | #0071e3 | 标题、关键数据、CTA |
| **背景深色** | #1a1a2e | 科技感配图背景 |
| **荧光强调** | #00d4ff | 高亮关键部位 |
| **警告/异常** | #E91E63 | 标注问题/风险 |
| **正常/成功** | #4CAF50 | 标注改善/达标 |
| **文字浅色** | #e6edf3 | 深色背景上文字 |

## 五、质量检查清单

- [ ] 配图是否回答了正文提出的具体问题？
- [ ] 科学内容是否准确（非美观性误导）？
- [ ] 风格是否与品牌视觉一致？
- [ ] AI提示词是否包含负面排除项？
- [ ] 画幅比例是否与预期平台一致？
- [ ] 关键数据/标注是否可读？
