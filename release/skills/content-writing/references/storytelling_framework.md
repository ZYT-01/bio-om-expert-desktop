# B端叙事框架

> 用途: content-writing 技能 · 参考文档
> 适用场景: 构建有说服力的B端内容叙事结构

## 一、四大叙事母题

### 母题1：英雄之旅（Hero's Journey）
应用于：客户案例、技术突破故事

```
平凡世界（行业现状/痛点）
    ↓
冒险召唤（发现新技术/新方案）
    ↓
拒绝召唤（客户犹豫：太新/太贵/太复杂）
    ↓
遇见导师（技术专家/成功案例）
    ↓
跨越门槛（首次尝试/小规模验证）
    ↓
考验与收获（数据验证/效果确认）
    ↓
满载而归（分享成功经验/成为案例）
```

### 母题2：问题-解决（Problem-Solution）
应用于：技术解析、产品介绍

```
现有问题（量化痛苦）
    ↓
根因分析（为什么现有方案不行）
    ↓
突破方案（我们的方法）
    ↓
效果验证（数据对比）
    ↓
价值总结（对客户的意义）
```

### 母题3：趋势洞察（Insight-Driven）
应用于：行业趋势、思想领导力

```
反常识观点（挑战既有认知）
    ↓
数据支撑（为什么这个观点是对的）
    ↓
趋势推演（这意味着什么）
    ↓
先机建议（现在该做什么）
```

### 母题4：对比评估（Comparison）
应用于：选型指南、技术评估

```
评估框架（我们用什么标准衡量）
    ↓
方案A分析（优势/局限）
    ↓
方案B分析（优势/局限）
    ↓
综合评分与建议（适用场景建议）
```

## 二、叙事节奏控制

| 阶段 | 占比 | 节奏 | 技巧 |
|------|------|------|------|
| 开场 | 15% | 快 | 一句话抓住注意力 |
| 铺垫 | 20% | 中 | 数据+场景建立可信度 |
| 高潮 | 40% | 慢 | 关键技术解析，深挖细节 |
| 收尾 | 25% | 快 | 快速总结+行动号召 |

## 三、金句生成公式

### 公式1：反常识公式
```
"大多数人以为X，但实际上Y"
示例："大多数人以为酸性pH可以分离空壳，但最新研究发现，碱性条件反而更精准。"
```

### 公式2：数字锚定公式
```
"{数字} + {时间/比例} + {结果}"
示例："一个参数调整，空壳率从15%降到3%。这个参数就是pH。"
```

### 公式3：类比破壁公式
```
"{专业概念} 就像 {日常事物}——{核心相似点}"
示例："病毒载体的质量控制，就像快递分拣——你要确保每个包裹里都有货。"
```

### 公式4：代价反射公式
```
"每{时间单位}，{决策延迟/不行动}就让你损失{可量化损失}"
示例："每延迟一个月解决空壳率问题，一条GMP产线就损失约200万元。"
```

## 四、叙事禁忌

- ❌ 从公司/产品角度叙事（"我们做了什么"）
- ✅ 从客户痛点/行业问题角度叙事（"你的困境是什么"）
- ❌ 线性罗列功能（"我们有A、B、C三个功能"）
- ✅ 因果链接功能（"因为A问题，所以需要B方案，效果是C"）
- ❌ 模糊的价值主张（"提升效率"）
- ✅ 量化的价值主张（"检测通量提升40%，每年节省800工时"）
