# Skill: report-generator（统一报告生成）

## 用途
接收来自三个研究技能的原始数据，按照标准模板生成格式统一、结构完整的Markdown研究报告。

## 触发条件
- 由 `web-research`、`url-research`、`local-research` 在完成数据采集后自动调用
- 也可独立调用：`/report-generator --data "data.json" --template "template.md"`

## 输入字段
| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| raw_data | object | 是 | 研究技能输出的原始数据 |
| topic | string | 是 | 报告主题 |
| template | string | 否 | 报告模板选择（默认标准模板） |
| source_type | enum | 是 | web / url / local |
